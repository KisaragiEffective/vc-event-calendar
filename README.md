# Virtual Communication event calendar

## 収録基準 (暫定)
* 日本の法令に違反するようなイベントは収録しません。
* 不特定多数の参加を目的としていないイベントは収録しません。
* 以下のプラットフォームに当てはまらないプラットフォームで開催されるイベントは収録しません。
  * VRChat
  * Cluster
  * ChilloutVR
  * NeosVR
  * Second Life
